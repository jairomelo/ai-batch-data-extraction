# AI Data Extraction Pipeline

This is an experiment to explore the possibilities for a pipeline to extract and structure data from archival or printed sources, using AI in combination with traditional scripting to reduce the number of tokens, prepare the attachments for easy parsing for the models.

The structure is based on OpenAI-compatible APIs, so it's possible to use any common provider, such as OpenAI, Anthropic, Meta, etc., and also custom providers such as universities or research centers, or can even run local small models without compromising the data with external services.

This pipeline is a simple exercise with a low level of reproducibility, mostly a personal project, but any suggestion is always welcomed, as well as any implementation beyond this project.
